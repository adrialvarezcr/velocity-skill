## Key Principles

1. **Imperfect beats perfect.** Launch at 70%, iterate from feedback.
2. **Movement beats analysis.** You learn by doing, not by planning.
3. **Psychology before strategy.** Fix your mindset first; strategy follows.
4. **Anti-patterns as warnings.** Know what kills ventures; avoid it.
5. **Threshold recognition.** Some moments don't repeat. Cross when they arrive.
